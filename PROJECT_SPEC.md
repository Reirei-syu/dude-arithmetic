# 项目规格说明 (PROJECT_SPEC.md)

## 项目名称
dude-arithmetic

## 项目描述
一个基于 React + Vite 的单页工具，用于随机生成加减乘除数学练习题，支持 A4 纸打印预览。

## 技术栈
- React 19
- Vite 6
- Tailwind CSS 4
- Lucide React (图标)
- Motion (动画)

## 核心功能
1. 选择数字 A 的位数 (1-5位)
2. 选择运算符号 (+, -, ×, ÷)
3. 选择数字 C 的位数 (1-5位)
4. 设置答案上限 D
5. 随机生成 50 道题目
6. A4 纸打印预览及打印功能
