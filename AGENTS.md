# 项目架构与协作指南 (AGENTS.md)

## 1. 架构原则
- **前端驱动**：这是一个纯静态的前端应用，不涉及后端数据库。
- **UI → Service → Core**：遵循分层约束，UI 层负责展示，逻辑由 React Hooks 或纯函数管理。
- **静态资源管理**：使用 Vite 处理所有静态资源，确保路径在 GitHub Pages 下正常工作。

## 2. 开发规范
- **Tailwind CSS 4**：使用最新版 Tailwind CSS 进行样式开发。
- **TypeScript**：严格遵循类型检查。
- **打印适配**：确保 CSS 中有正确的 `@media print` 样式（或通过 js 生成打印窗口）。

## 3. 部署规范
- **自动化构建**：所有部署必须通过 GitHub Actions 进行。
- **分支管理**：`main` 分支作为开发和部署来源。

## 4. 特殊约束
- 页面必须支持 A4 打印适配。
- 所有 UI 文案使用简体中文。
