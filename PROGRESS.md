## 本次修改

- 在主页面新增“出题数量”输入，支持自定义题目总数
- 新增“打印独立答案页”开关

- 打印逻辑改为按页拆分题目页，并在答案页启用强制分页
- 预览弹窗增加页数摘要，明确题目页和答案页数量

## 影响范围

## 验证结果

- `npm run lint`：通过
- `npm run build`：通过
- 临时 Playwright 真机尺寸验证：通过
- 验证结论：
  - 可输入出题数量
  - 开启答案页后，打印弹窗中存在独立的 `.answer-page`
  - 80 题场景下，打印内容自动拆分为 4 页，其中答案页 2 页，未与题目页混排

## 风险备注

- 本次没有改动部署逻辑
- 题目页分页是按字号和列数做近似计算，极端组合下分页密度可能与实际打印略有差异，但答案页会始终从新页开始

## Lessons Learned

- 一旦允许自定义题量，打印逻辑就必须从“单页缩放”升级为“分页输出”
- 答案页需求本质上是分页规则，不只是多打一块内容
